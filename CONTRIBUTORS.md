# Contributors

ContentOps powered by SecM8 is built on the contributions of the
people listed below.

This file is updated by hand when a contributor opens their first
merged PR. To add yourself, open a PR with your GitHub handle (and
optionally your real name) appended in alphabetical order.

Contribution counts and timestamps are visible via `git log` and the
GitHub contributors view; this file exists to surface the human-readable
roster and to satisfy the "all contributors" pattern many enterprise
adopters look for.

## Project contributors

- @KustoKing — initial author and lead maintainer

## How contributions are recorded

Every commit is attributed in `git log` to its author. This file is
an additional, optional roster. Listing yourself here is not required
for your contribution to count, and not listing yourself does not
diminish the contribution.
